# Deep Learning — Laboratory Sessions

ESI-SBA · AI & Data Science Engineering · 2024/2025

## Overview

Laboratory sessions for the Deep Learning module at ESI-SBA, covering foundational theory through advanced architectures including sequential models, CNNs, and transfer learning for medical image classification.

## Laboratory Sessions

| Lab | Topics |
|-----|--------|
| TP-1 | Perceptrons, Activation Functions, Backpropagation |
| TP-2 | Regularization, Dropout, Batch Normalization, Optimizers |
| TP-3 | Sequential Models — LSTM, GRU, MNIST, CIFAR-10 |
| TP-4 | CNNs — Architectures, Pooling, Data Augmentation |
| TP-5 | Transfer Learning — EfficientNet and ShuffleNet for COVID-19 X-ray Classification |
